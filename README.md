# Indonesian Sign Language Detection

Sistem pengenalan Bahasa Isyarat Indonesia (BISINDO) berbasis YOLOv8.

## Fitur

- Deteksi 26 huruf alfabet BISINDO
- Deteksi 15 kosakata BISINDO
- Mobile application menggunakan Flutter
- REST API menggunakan FastAPI
- Penyimpanan data menggunakan PostgreSQL

## Teknologi

### Frontend
- Flutter

### Backend
- FastAPI
- SQLAlchemy

### AI Model
- YOLOv8

### Database
- PostgreSQL

## Struktur Project

backend/     → FastAPI API + YOLO inference
frontend/    → Flutter mobile application
database/    → SQL database schema
